要使用GDB进行调试，要在编译的时候加上“-g”选项：
例：
?
View Code
BASH
1
2
$
gcc
-g
main.c
-o
main
$
gdb
main
以下是几个常用命令：
help:帮助命令，可以直接在help后加要查询的命令。
list:列出源代码，一次列十行。也可以在后面加上函数名查询函数。
start:开始调试，程序运行到main函数入口等待操作。
next(n):单步调试，不进入函数。
setp(s):单步调试，如有函数进入。
backtrack(bt):查看函数调用帧栈。
info locals:查看局部变量的值。
frame:选择帧栈。
print(p):打出指定变量值。
print(p) var=x:将变量var的值设置为x。
set var=x:功能同上。
finish:一直运行到当前函数返回为止。
